# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Orch8, please report it responsibly.

**Do not open a public issue.**

Email: security@orch8.io

We will acknowledge receipt within 48 hours and provide an initial assessment within 7 days.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Scope

The following are in scope:
- Orch8 engine binary (`orch8-server`)
- Orch8 CLI (`orch8`)
- Official SDKs (sdk-node, sdk-python, sdk-go)
- Helm chart (orch8-io/helm-charts)

The following are out of scope:
- Third-party integrations
- Issues in dependencies (report upstream)
